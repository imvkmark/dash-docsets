eio_sync

(PECL eio >= 0.0.1dev)

eio_sync — Commit buffer cache to disk

说明

eio_sync(int $pri = EIO_PRI_DEFAULT, callable $callback = NULL, mixed $data = NULL): resource

参数

此函数没有参数。

返回值

eio_sync() returns request resource on success, 或者在失败时返回 false.
