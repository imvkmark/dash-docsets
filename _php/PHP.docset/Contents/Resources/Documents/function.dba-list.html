dba_list

(PHP 4 >= 4.3.0, PHP 5, PHP 7, PHP 8)

dba_list — List all open database files

说明

dba_list(): array

dba_list() list all open database files.

参数

此函数没有参数。

返回值

An associative array, in the form resourceid => filename.
