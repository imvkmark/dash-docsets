OAuth::enableRedirects

(PECL OAuth >= 0.99.9)

OAuth::enableRedirects — 启用重定向

说明

public OAuth::enableRedirects(): bool

启用自动重定向，此项默认为启用。

警告

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

true

参见

  • OAuth::disableRedirects() - 关闭重定向
