Ds\Sequence::get

(PECL ds >= 1.0.0)

Ds\Sequence::get — Returns the value at a given index

说明

abstract public Ds\Sequence::get(int $index): mixed

Returns the value at a given index.

参数

index

The index to access, starting at 0.

返回值

The value at the requested index.

错误／异常

OutOfRangeException if the index is not valid.

范例

示例 #1 Ds\Sequence::get() example

<?php
$sequence = new \Ds\Vector(["a", "b", "c"]);

var_dump($sequence->get(0));
var_dump($sequence->get(1));
var_dump($sequence->get(2));
?>

以上例程的输出类似于：

string(1) "a"
string(1) "b"
string(1) "c"

示例 #2 Ds\Sequence::get() example using array syntax

<?php
$sequence = new \Ds\Vector(["a", "b", "c"]);

var_dump($sequence[0]);
var_dump($sequence[1]);
var_dump($sequence[2]);
?>

以上例程的输出类似于：

string(1) "a"
string(1) "b"
string(1) "c"
