openssl_free_key

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

openssl_free_key — 释放密钥资源

说明

openssl_free_key(resource $key_identifier): void

openssl_free_key() 从内存中释放和指定的 key_identifier相关联的密钥。

参数

key_identifier

返回值

没有返回值。
