ldap_rename_ext

(PHP 7 >= 7.3.0, PHP 8)

ldap_rename_ext — Modify the name of an entry

说明

ldap_rename_ext(
    LDAP\Connection $ldap,
    string $dn,
    string $new_rdn,
    string $new_parent,
    bool $delete_old_rdn,
    ?array $controls = null
): LDAP\Result|false

Does the same thing as ldap_rename() but returns an LDAP\Result instance to be parsed with ldap_parse_result().

参数

See ldap_rename()

返回值

Returns an LDAP\Result instance, 或者在失败时返回 false.

更新日志

版本 说明
8.1.0 现在 ldap 参数接受 LDAP\Connection 实例，之前接受 资源(resource)。
8.1.0 Returns an LDAP\Result instance now; previously, a 资源(resource) was returned.
8.0.0 controls is nullable now; previously, it defaulted to [].
7.3 Support for controls added
