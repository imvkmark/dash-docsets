DirectoryIterator::__construct

(PHP 5, PHP 7, PHP 8)

DirectoryIterator::__construct — Constructs a new directory iterator from a path

说明

public DirectoryIterator::__construct(string $directory)

Constructs a new directory iterator from a path.

参数

directory

The path of the directory to traverse.

错误／异常

Throws an UnexpectedValueException if the directory cannot be opened.

Throws a RuntimeException if the directory is an empty string.

范例

示例 #1 A DirectoryIterator::__construct() example

This example will list the contents of the directory containing the script.

<?php
$dir = new DirectoryIterator(dirname(__FILE__));
foreach ($dir as $fileinfo) {
    if (!$fileinfo->isDot()) {
        var_dump($fileinfo->getFilename());
    }
}
?>
