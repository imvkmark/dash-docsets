is_finite

(PHP 4 >= 4.2.0, PHP 5, PHP 7, PHP 8)

is_finite — 判断是否为有限值

说明

is_finite(float $val): bool

检查 val 是否是是本机平台上浮点数所允许范围中的一个合法的有限值。

参数

val

要检查的值

返回值

如果 val 是本机平台上 PHP 浮点数所允许范围中的一个合法的有限值，则返回 true。

参见

  • is_infinite() - 判断是否为无限值
  • is_nan() - 判断是否为合法数值
