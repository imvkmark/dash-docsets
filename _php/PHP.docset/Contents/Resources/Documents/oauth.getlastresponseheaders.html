OAuth::getLastResponseHeaders — 获取最后一次响应的头信息

说明

public OAuth::getLastResponseHeaders(): string

获取最后一次响应的头信息。

参数

此函数没有参数。

返回值

返回一个包含最后一次响应头信息的字符串 或者在失败时返回 false 。
