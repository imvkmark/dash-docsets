imagerectangle

(PHP 4, PHP 5, PHP 7, PHP 8)

imagerectangle — 画一个矩形

说明

imagerectangle(
    resource $image,
    int $x1,
    int $y1,
    int $x2,
    int $y2,
    int $col
): bool

imagerectangle() 用 col 颜色在 image 图像中画一个矩形，其左上角坐标为 x1, y1，右下角坐标为 x2, y2。图像的左上角坐标为 0, 0。
