is_tainted

(PECL taint >=0.1.0)

is_tainted — Checks whether a string is tainted

说明

is_tainted(string $string): bool

Checks whether a string is tainted

参数

string

返回值

Return TRUE if the string is tainted, FALSE otherwise.
