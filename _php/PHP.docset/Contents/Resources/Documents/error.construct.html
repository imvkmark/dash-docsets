Error::__construct

(PHP 7, PHP 8)

Error::__construct — 初始化 error 对象

说明

public Error::__construct(string $message = "", int $code = 0, ?Throwable $previous = null)

初始化 Error。

参数

message

错误信息。

code

错误代码。

previous

先前的 throwable，用于异常链。

注释

注意:

message 不是二进制安全的。
