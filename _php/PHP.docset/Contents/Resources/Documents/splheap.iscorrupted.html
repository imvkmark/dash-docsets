SplHeap::isCorrupted

(PHP 7, PHP 8)

SplHeap::isCorrupted — Tells if the heap is in a corrupted state

说明

public SplHeap::isCorrupted(): bool

参数

此函数没有参数。

返回值

Returns true if the heap is corrupted, false otherwise.
