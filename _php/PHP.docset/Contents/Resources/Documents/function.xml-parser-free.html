xml_parser_free

(PHP 4, PHP 5, PHP 7, PHP 8)

xml_parser_free — 释放指定的 XML 解析器

描述

xml_parser_free(resource $parser): bool

parser
指向要释放的 XML 解析器的指针。

如果 parser 参数没有指向一个合法的解析器，该函数将返回 false，否则将释放指定的解析器并返回 true。
