(PECL hrtime >= 0.6.0)

HRTime\PerformanceCounter::getTicks — Current ticks from the system

说明

public static HRTime\PerformanceCounter::getTicks(): int

Returns the ticks count.

参数

此函数没有参数。

返回值

Returns int indicating the ticks count.
