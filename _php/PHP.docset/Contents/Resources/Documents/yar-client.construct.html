Yar_Client::__construct

(PECL yar >= 1.0.0)

Yar_Client::__construct — 创建一个客户端实例

说明

final public Yar_Client::__construct(string $url)

创建一个Yar_Client 到一个 Yar_Server的实例.

参数

url

服务端的HTTP URL路径.

返回值

Yar_Client实例.

范例

示例 #1 Yar_Client::__construct()示例

<?php
$client = new Yar_Client("http://host/api/");
?>
