mcrypt_enc_self_test

(PHP 4 >= 4.0.2, PHP 5, PHP 7 < 7.2.0, PECL mcrypt >= 1.0.0)

mcrypt_enc_self_test — 在打开的模块上进行自检

警告

本函数已自 PHP 7.1.0 起废弃。强烈建议不要使用本函数。

说明

mcrypt_enc_self_test(resource $td): int

在 td 指定的算法 上进行自检操作。

参数

td

加密描述符。

返回值

自检成功返回 0，失败则返回一个小于零的int。
