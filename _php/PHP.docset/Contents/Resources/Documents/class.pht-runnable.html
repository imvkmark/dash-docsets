The Runnable interface

(PECL pht >= 0.0.1)

简介

The pht\Runnable interface enforces the implementation of a run() method on classes that should be threaded. This method acts as the entry point of the threaded class.

接口摘要

class pht\Runnable {
/* 方法 */
public run(): void
}
