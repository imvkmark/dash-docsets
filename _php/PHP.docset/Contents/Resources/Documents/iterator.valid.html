Iterator::valid

(PHP 5, PHP 7, PHP 8)

Iterator::valid — 检查当前位置是否有效

说明

public Iterator::valid(): bool

此方法在 Iterator::rewind() 和 Iterator::next() 方法之后被调用以此用来检查当前位置是否有效。

参数

此函数没有参数。

返回值

返回将被转换为bool 。成功时返回 true， 或者在失败时返回 false。
