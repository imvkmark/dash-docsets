Ds\Sequence::reverse

(PECL ds >= 1.0.0)

Ds\Sequence::reverse — Reverses the sequence in-place

说明

abstract public Ds\Sequence::reverse(): void

Reverses the sequence in-place.

参数

此函数没有参数。

返回值

没有返回值。

范例

示例 #1 Ds\Sequence::reverse() example

<?php
$sequence = new \Ds\Vector(["a", "b", "c"]);
$sequence->reverse();

print_r($sequence);
?>

以上例程的输出类似于：

Ds\Vector Object
(
    [0] => c
    [1] => b
    [2] => a
)
