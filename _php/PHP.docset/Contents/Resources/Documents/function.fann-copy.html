fann_copy

(PECL fann >= 1.0.0)

fann_copy — 创建一个 fann 结构体的副本。

说明

fann_copy(resource $ann): resource

创建一个 fann 结构体的副本。

参数

ann

神经网络 资源。

返回值

成功，返回神经网络资源的副本，失败则返回false
