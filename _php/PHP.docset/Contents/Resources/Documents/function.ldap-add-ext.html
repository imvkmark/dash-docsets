ldap_add_ext

(PHP 7 >= 7.3.0, PHP 8)

ldap_add_ext — Add entries to LDAP directory

说明

ldap_add_ext(
    LDAP\Connection $ldap,
    string $dn,
    array $entry,
    ?array $controls = null
): LDAP\Result|false

Does the same thing as ldap_add() but returns an LDAP\Result instance to be parsed with ldap_parse_result().

参数

See ldap_add()

返回值

Returns an LDAP\Result instance, 或者在失败时返回 false.

更新日志

版本 说明
8.1.0 现在 ldap 参数接受 LDAP\Connection 实例，之前接受 资源(resource)。
8.1.0 Returns an LDAP\Result instance now; previously, a 资源(resource) was returned.
8.0.0 controls is nullable now; previously, it defaulted to [].

注释

注意: 此函数可安全用于二进制对象。

参见

  • ldap_parse_result() - Extract information from result
