array_key_last

(PHP 7 >= 7.3.0, PHP 8)

array_key_last — 获取一个数组的最后一个键值

说明

array_key_last(array $array): int|string|null

取得指定数组的 array 最后一个键值，不会影响到原数组的内部指针。

参数

array

要操作的数组。

返回值

返回 array 的最后一个键值（如果不为空），否则返回 null。

参见

  • array_key_first() - 获取指定数组的第一个键
  • end() - 将数组的内部指针指向最后一个单元
