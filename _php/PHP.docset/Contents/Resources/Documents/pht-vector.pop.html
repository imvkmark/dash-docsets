pht\Vector::pop

(PECL pht >= 0.0.1)

pht\Vector::pop — Pops a value to the vector

说明

public pht\Vector::pop(): mixed

This method pops a value from the end of a vector (in constant time). Popping a value from an empty vector will result in an Error exception.

参数

此函数没有参数。

返回值

The value from the end of the vector.

范例

示例 #1 Popping a value from a vector

<?php

use pht\Vector;

$vector = new Vector();

$vector->push(1);
$vector[] = 2;

var_dump($vector->pop());

以上例程会输出：

int(2)
