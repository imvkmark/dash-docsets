Yaf_Application::run

(Yaf >=1.0.0)

Yaf_Application::run — 运行 Yaf_Application

说明

public Yaf_Application::run(): void

运行一个Yaf_Application，开始接受并处理请求，分发路由，做出相应的响应。最终将响应返回给客户端。

参数

此函数没有参数。

返回值
