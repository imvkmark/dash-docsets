ReflectionType 类

(PHP 7, PHP 8)

简介

ReflectionType 类用于获取函数、类方法的参数或者返回值的类型。 反射扩展声明了以下子类型：

  • ReflectionNamedType (自 PHP 7.1.0 起)
  • ReflectionUnionType (自 PHP 8.0.0 起)

类摘要

abstract class ReflectionType {
/* 方法 */
public allowsNull(): bool
public __toString(): string
}
