OAuth::disableRedirects

(PECL OAuth >= 0.99.9)

OAuth::disableRedirects — 关闭重定向

说明

public OAuth::disableRedirects(): bool

禁用自动跟随其后的重定向，这样，手动重定向请求。

警告

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

true
