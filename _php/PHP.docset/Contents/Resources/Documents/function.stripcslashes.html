stripcslashes

(PHP 4, PHP 5, PHP 7, PHP 8)

stripcslashes — 反引用一个使用 addcslashes() 转义的字符串

说明

stripcslashes(string $str): string

返回反转义后的字符串。可识别类似 C 语言的 \n，\r，... 八进制以及十六进制的描述。

参数

str

需要反转义的字符串。

返回值

返回反转义后的字符串。
