ReflectionClass::setStaticPropertyValue

(PHP 5 >= 5.1.2, PHP 7, PHP 8)

ReflectionClass::setStaticPropertyValue — 设置静态属性的值

说明

public ReflectionClass::setStaticPropertyValue(string $name, string $value): void

设置静态属性的值。

警告

本函数还未编写文档，仅有参数列表。

参数

name

属性的名称。

value

属性的值。

返回值

没有返回值。
