LimitIterator::valid

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

LimitIterator::valid — Check whether the current element is valid

说明

public LimitIterator::valid(): bool

Checks whether the current element is valid.

参数

此函数没有参数。

返回值

成功时返回 true， 或者在失败时返回 false。
