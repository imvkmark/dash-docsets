fann_reset_MSE

(PECL fann >= 1.0.0)

fann_reset_MSE — 重置网络中的均方误差。

说明

fann_reset_MSE(string $ann): bool

重置网络中的均方误差。

该函数也会重置失败时位的数量。

参数

ann

神经网络 资源。

返回值

成功时返回 true，其它情况下返回 false。

参见

  • fann_get_MSE() - 从网络中读取均方误差。
  • fann_get_bit_fail() - 失败位的数量
  • fann_get_bit_fail_limit() - 返回训练期间使用的误差限制
