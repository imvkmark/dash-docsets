ibase_free_result

(PHP 5, PHP 7 < 7.4.0)

ibase_free_result — Free a result set

说明

ibase_free_result(resource $result_identifier): bool

Frees a result set.

参数

result_identifier

A result set created by ibase_query() or ibase_execute().

返回值

成功时返回 true， 或者在失败时返回 false。
