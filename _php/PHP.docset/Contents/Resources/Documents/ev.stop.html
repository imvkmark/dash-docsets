Ev::stop

(PECL ev >= 0.2.0)

Ev::stop — Stops the default event loop

说明

final public static Ev::stop( int $how = ?): void

Stops the default event loop

参数

how

One of Ev::BREAK_* constants .

返回值

没有返回值。
