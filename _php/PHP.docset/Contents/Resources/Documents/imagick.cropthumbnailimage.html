Imagick::cropThumbnailImage

(PECL imagick 2, PECL imagick 3)

Imagick::cropThumbnailImage — 创建缩略图

说明

Imagick::cropThumbnailImage(int $width, int $height): bool

首先将图片放大或者缩小，然后从图片的中间裁剪到指定的的大小。

参数

width

缩略图的宽

height

缩略图的高

返回值

成功时返回 true。

错误／异常

错误时抛出 ImagickException。
