FilterIterator 类

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

简介

这个抽象类的遍历并过滤出不想要的值。这个类应该被实现了迭代过滤器的类继承 FilterIterator::accept() 方法必须被子类实现。
