trader_tsf

(PECL trader >= 0.2.0)

trader_tsf — Time Series Forecast

说明

trader_tsf(array $real, int $timePeriod = ?): array

参数

real

浮点数数组。

timePeriod

Number of period. Valid range from 2 to 100000.

返回值

Returns an array with calculated data or false on failure.
