Serializable::unserialize

(PHP 5 >= 5.1.0, PHP 7, PHP 8)

Serializable::unserialize — 构造对象

说明

public Serializable::unserialize(string $data): void

在反序列化对象时被调用。

注意:

这个方法担当着对象构造器的角色。在此方法之后，__construct() 将不会被调用。

参数

data

对象的字符串表示

返回值

返回没有序列化之前的原始值。

参见

  • __wakeup()
  • __unserialize()
