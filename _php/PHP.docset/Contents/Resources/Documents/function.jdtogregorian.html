(PHP 4, PHP 5, PHP 7, PHP 8)

JDToGregorian — 转变一个Julian Day计数为Gregorian历法日期

说明

jdtogregorian(int $julianday): string

转变一个julian天数为gregorian历法的“月/日/年”形式的日期

参数

julianday

一个julian天数

返回值

“月/日/年”形式的gregorian日期
