Imagick::removeImage

(PECL imagick 2, PECL imagick 3)

Imagick::removeImage — Removes an image from the image list

说明

public Imagick::removeImage(): bool

Removes an image from the image list.

参数

此函数没有参数。

返回值

成功时返回 true。

错误／异常

错误时抛出 ImagickException。
