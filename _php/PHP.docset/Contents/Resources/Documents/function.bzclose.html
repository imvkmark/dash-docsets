bzclose

(PHP 4 >= 4.0.4, PHP 5, PHP 7, PHP 8)

bzclose — 关闭一个 bzip2 文件

说明

bzclose(resource $bz): bool

关闭给出的 bzip2 文件指针。

参数

bz

文件指针。它必须是有效的并且指向 bzopen() 成功打开的文件。

返回值

成功时返回 true， 或者在失败时返回 false。

  • bzopen() - 打开 bzip2 压缩文件
