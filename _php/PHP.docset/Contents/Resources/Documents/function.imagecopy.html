imagecopy

(PHP 4, PHP 5, PHP 7, PHP 8)

imagecopy — 拷贝图像的一部分

说明

imagecopy(
    resource $dst_im,
    resource $src_im,
    int $dst_x,
    int $dst_y,
    int $src_x,
    int $src_y,
    int $src_w,
    int $src_h
): bool

将 src_im 图像中坐标从 src_x，src_y 开始，宽度为 src_w，高度为 src_h 的一部分拷贝到 dst_im 图像中坐标为 dst_x 和 dst_y 的位置上。
