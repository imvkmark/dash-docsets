fann_set_output_scaling_params

(PECL fann >= 1.0.0)

fann_set_output_scaling_params — 根据训练数据计算将来使用的输出缩放参数

说明

fann_set_output_scaling_params(
    resource $ann,
    resource $train_data,
    float $new_output_min,
    float $new_output_max
): bool

根据训练数据计算将来使用的输出缩放参数

参数

ann

神经网络 资源。

train_data

神经网络训练数据 资源。

new_output_min

缩放后输出数据的期望下限(不严格遵循)

new_output_max

缩放后输出数据的期望上限(不严格遵循)

返回值

成功时返回 true，其它情况下返回 false。
