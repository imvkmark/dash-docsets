asinh

(PHP 4 >= 4.1.0, PHP 5, PHP 7, PHP 8)

asinh — 反双曲正弦

说明

asinh(float $arg): float

返回 arg 的反双曲正弦值，即，其双曲正弦为 arg 的那个值。

参数

arg

要处理的参数

返回值

返回 arg 的反双曲正弦值

版本 说明
