read_exif_data

(PHP 4 >= 4.0.1, PHP 5, PHP 7)

read_exif_data — 别名 exif_read_data()

警告

This alias was DEPRECATED in PHP 7.2.0, and REMOVED as of PHP 8.0.0.

说明

此函数是该函数的别名： exif_read_data().
