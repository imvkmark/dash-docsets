Throwable::getTraceAsString

(PHP 7, PHP 8)

Throwable::getTraceAsString — 以 string 形式获取堆栈踪迹

说明

public Throwable::getTraceAsString(): string

参数

此函数没有参数。

返回值

返回 string 形式的堆栈踪迹（Stack Trace）。
