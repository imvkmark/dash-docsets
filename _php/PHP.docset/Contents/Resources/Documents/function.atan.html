atan

(PHP 4, PHP 5, PHP 7, PHP 8)

atan — 反正切

说明

atan(float $arg): float

返回 arg 的反正切值，单位是弧度。atan() 是 tan() 的反函数，它的意思是在 atan() 范围里的每个值都是 a==tan(atan(a))。

参数

arg

要处理的参数

返回值

arg 的反正切弧度。

参见

  • tan() - 正切
  • atanh() - 反双曲正切
  • asin() - 反正弦
  • acos() - 反余弦
