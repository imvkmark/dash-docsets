timezone_version_get

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

timezone_version_get — 获取 timezonedb 的版本

说明

timezone_version_get(): string

返回当前 timezonedb 的版本。

返回值

返回一个 string。

范例

示例 #1 获取 timezonedb 版本

<?php
echo timezone_version_get();
?>

以上例程的输出类似于：

2009.7
