realpath_cache_size

(PHP 5 >= 5.3.2, PHP 7, PHP 8)

realpath_cache_size — 获取真实路径缓冲区的大小

说明

realpath_cache_size(): int

获取真实路径缓存区大小在内存中的使用量。

返回值

返回真实路径缓存区使用内存的用量。

范例

示例 #1 realpath_cache_size() example

<?php
var_dump(realpath_cache_size());
?>

以上例程的输出类似于：

int(412)
