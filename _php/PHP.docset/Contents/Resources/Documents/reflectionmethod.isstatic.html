ReflectionMethod::isStatic

(PHP 5, PHP 7, PHP 8)

ReflectionMethod::isStatic — 判断方法是否是静态方法

说明

public ReflectionMethod::isStatic(): bool

判断方法是否是静态方法

参数

此函数没有参数。

返回值

静态方法返回 true，否则返回 false
