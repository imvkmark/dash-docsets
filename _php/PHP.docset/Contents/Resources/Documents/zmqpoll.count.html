ZMQPoll::count

(PECL zmq >= 0.5.0)

ZMQPoll::count — Count items in the poll set

说明

public ZMQPoll::count(): int

Count the items in the poll set.

参数

此函数没有参数。

返回值

Returns an int representing the amount of items in the poll set.
