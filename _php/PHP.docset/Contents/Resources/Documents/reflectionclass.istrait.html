ReflectionClass::isTrait

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

ReflectionClass::isTrait — 返回了是否为一个 trait

说明

public ReflectionClass::isTrait(): bool

警告

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

如果是 trait 返回 true，否则返回 false。 在出现错误的情况下，将会返回 null。
