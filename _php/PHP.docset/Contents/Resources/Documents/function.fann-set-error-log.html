fann_set_error_log

(PECL fann >= 1.0.0)

fann_set_error_log — 设置错误记录保存的位置。

说明

fann_set_error_log(resource $errdat, string $log_file): void

设置错误记录保存的位置。

参数

errdat

Either neural network resource or neural network trainining data resource.

log_file

日志文件的路径。

返回值

无返回值。
