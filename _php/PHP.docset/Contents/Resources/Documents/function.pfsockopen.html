pfsockopen

(PHP 4, PHP 5, PHP 7, PHP 8)

pfsockopen — 打开一个持久的网络连接或者Unix套接字连接。

说明

pfsockopen(
    string $hostname,
    int $port = -1,
    int &$errno = ?,
    string &$errstr = ?,
    float $timeout = ini_get("default_socket_timeout")
): resource

这个函数的作用与fsockopen()完全一样的，不同的地方在于当在脚本执行完后，连接一直不会关闭。可以说它是fsockopen()的长连接版本。

参数

对于其参数的信息，请参考fsockopen()的文档。
