EvWatcher::stop

(PECL ev >= 0.2.0)

EvWatcher::stop — Stops the watcher

说明

public EvWatcher::stop(): void

Marks the watcher as inactive. Note that only active watchers will receive events.

参数

此函数没有参数。

返回值

没有返回值。
