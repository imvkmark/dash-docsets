trader_typprice

(PECL trader >= 0.2.0)

trader_typprice — Typical Price

说明

trader_typprice(array $high, array $low, array $close): array

参数

high

高价，浮点数数组。

low

低价，浮点数数组。

close

收盘价，浮点数数组。

返回值

Returns an array with calculated data or false on failure.
