The StompFrame class

(PECL stomp >= 0.1.0)

简介

Represents a message which was sent or received from a Stomp compliant Message Broker.

类摘要

class StompFrame {
/* 属性 */
public $command;
public $headers;
public $body;
/* 方法 */
__construct(string $command = ?, array $headers = ?, string $body = ?)
}

属性

command

Frame command.

headers

Frame headers (array).

body

Frame body.
