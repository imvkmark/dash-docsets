MongoDB\Driver\ReadConcern::getLevel

(mongodb >=1.0.0)

MongoDB\Driver\ReadConcern::getLevel — Returns the ReadConcern's "level" option

说明

final public MongoDB\Driver\ReadConcern::getLevel(): ?string

参数

此函数没有参数。

返回值

Returns the ReadConcern's "level" option.

错误／异常

  • Throws MongoDB\Driver\Exception\InvalidArgumentException on argument parsing errors.

范例

示例 #1 MongoDB\Driver\ReadConcern::getLevel() example

<?php

$rc = new MongoDB\Driver\ReadConcern();
var_dump($rc->getLevel());

$rc = new MongoDB\Driver\ReadConcern(MongoDB\Driver\ReadConcern::LOCAL);
var_dump($rc->getLevel());

$rc = new MongoDB\Driver\ReadConcern(MongoDB\Driver\ReadConcern::MAJORITY);
var_dump($rc->getLevel());

?>

以上例程会输出：

NULL
string(5) "local"
string(8) "majority"
