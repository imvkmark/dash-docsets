ReflectionClass::getModifiers

(PHP 5, PHP 7, PHP 8)

ReflectionClass::getModifiers — 获取类的修饰符

说明

public ReflectionClass::getModifiers(): int

返回这个类访问修饰符的位字段。

参数

此函数没有参数。

返回值

返回 修饰符常量 的位掩码。

参见

  • ReflectionClass::getProperties() - 获取一组属性
  • Reflection::getModifierNames() - 获取修饰符的名称
