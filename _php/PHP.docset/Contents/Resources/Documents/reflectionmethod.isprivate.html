(PHP 5, PHP 7, PHP 8)

ReflectionMethod::isPrivate — 判断方法是否是私有方法

说明

public ReflectionMethod::isPrivate(): bool

判断方法是否是私有方法

警告

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

私有方法返回 true，否则返回 false
