trader_mama

(PECL trader >= 0.2.0)

trader_mama — MESA Adaptive Moving Average

说明

trader_mama(array $real, float $fastLimit = ?, float $slowLimit = ?): array

参数

real

浮点数数组。

fastLimit

Upper limit use in the adaptive algorithm. Valid range from 0.01 to 0.99.

slowLimit

Lower limit use in the adaptive algorithm. Valid range from 0.01 to 0.99.

返回值

Returns an array with calculated data or false on failure.
