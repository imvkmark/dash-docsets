ssh2_auth_agent

(PECL ssh2 >= 0.12)

ssh2_auth_agent — Authenticate over SSH using the ssh agent

说明

ssh2_auth_agent(resource $session, string $username): bool

Authenticate over SSH using the ssh agent

注意: The ssh2_auth_agent() function will only be available when the ssh2 extension is compiled with libssh >= 1.2.3.

参数

session

An SSH connection link identifier, obtained from a call to ssh2_connect().

username

Remote user name.

返回值

成功时返回 true， 或者在失败时返回 false。

范例

示例 #1 Authenticating with a ssh agent

<?php
$connection = ssh2_connect('shell.example.com', 22);

if (ssh2_auth_agent($connection, 'username')) {
  echo "Authentication Successful!\n";
} else {
  die('Authentication Failed...');
}
?>
