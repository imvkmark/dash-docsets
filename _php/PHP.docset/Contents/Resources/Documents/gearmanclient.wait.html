GearmanClient::wait

(PECL gearman >= 0.6.0)

GearmanClient::wait — Wait for I/O activity on all connections in a client

说明

public GearmanClient::wait(): bool

This waits for activity from any one of the connected servers.

参数

此函数没有参数。

返回值

true on success false on an error.
