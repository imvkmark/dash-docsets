__autoload

(PHP 5, PHP 7)

__autoload — 尝试加载未定义的类

警告

本函数已自 PHP 7.2.0 起被废弃，并自 PHP 8.0.0 起被移除。 强烈建议不要依赖本函数。

说明

__autoload(string $class): void

你可以通过定义这个函数来启用类的自动加载。

参数

class

待加载的类名。

返回值

没有返回值。

参见

  • spl_autoload_register() - 注册给定的函数作为 __autoload 的实现
