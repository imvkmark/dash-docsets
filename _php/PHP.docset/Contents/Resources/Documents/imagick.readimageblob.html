Imagick::readImageBlob

(PECL imagick 2, PECL imagick 3)

Imagick::readImageBlob — 从二进制字符串里读取图片

说明

Imagick::readImageBlob(string $image, string $filename = ?): bool

从二进制字符串里读取图片

参数

image

返回值

成功时返回 true。

错误／异常

错误时抛出 ImagickException。
