Ds\Pair::isEmpty — Returns whether the pair is empty

说明

public Ds\Pair::isEmpty(): bool

Returns whether the pair is empty.

参数

此函数没有参数。

返回值

Returns true if the pair is empty, false otherwise.

范例

示例 #1 Ds\Pair::isEmpty() example

<?php
$a = new \Ds\Pair("a", 1);
$b = new \Ds\Pair();

var_dump($a->isEmpty());
var_dump($b->isEmpty());
?>

以上例程的输出类似于：

bool(false)
bool(true)
