ftp_connect

(PHP 4, PHP 5, PHP 7, PHP 8)

ftp_connect — 建立一个新的 FTP 连接

说明

ftp_connect(string $host, int $port = 21, int $timeout = 90): resource

ftp_connect() 将会建立一个 FTP 连接到指定的服务器 host。

参数

host

要连接的服务器地址。此参数后面不应以斜线结尾，前面也不需要用 ftp:// 开头。

port

为要连接到的 FTP 器的端口号，如果没有设置或者为 0，则会使用默认的端口 21 来连接。

timeout

用来设置网络传输的超时时间限制。如果此选项留空，则默认的值为 90 秒。超时时间可以在任何时候通过函数 ftp_set_option() 及 ftp_get_option() 来改变和获取。

返回值

如果成功返回一个 FTP 连接资源句柄，发生错误则返回 false。

范例

示例 #1 ftp_connect() 示例

<?php

$ftp_server = "ftp.example.com";

// set up a connection or die
$conn_id = ftp_connect($ftp_server) or die("Couldn't connect to $ftp_server"); 

?>
