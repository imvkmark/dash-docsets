gc_enabled

(PHP 5 >= 5.3.0, PHP 7, PHP 8)

gc_enabled — 返回循环引用计数器的状态

说明

gc_enabled(): bool

返回循环引用计数器的状态。

参数

此函数没有参数。

返回值

如果垃圾收集器已启用则返回 true，否则返回 false。

范例

示例 #1 一个 gc_enabled() 例子

<?php
if(gc_enabled()) gc_collect_cycles();
?>
