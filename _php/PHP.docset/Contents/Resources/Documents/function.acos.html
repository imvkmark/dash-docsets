acos

(PHP 4, PHP 5, PHP 7, PHP 8)

acos — 反余弦

说明

acos(float $arg): float

返回 arg 的反余弦值，单位是弧度。acos() 是 cos() 的反函数，它的意思是在 acos() 范围里的每个值都是 a==cos(acos(a)) 。

参数

arg

要处理的参数

返回值

arg 的反余弦弧度。

参见

  • cos() - 余弦
  • acosh() - 反双曲余弦
  • asin() - 反正弦
  • atan() - 反正切
