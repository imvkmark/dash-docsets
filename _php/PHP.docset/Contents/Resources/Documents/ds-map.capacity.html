Ds\Map::capacity

(PECL ds >= 1.0.0)

Ds\Map::capacity — Returns the current capacity

说明

public Ds\Map::capacity(): int

Returns the current capacity.

参数

此函数没有参数。

返回值

The current capacity.

范例

示例 #1 Ds\Map::capacity() example

<?php
$map = new \Ds\Map();
var_dump($map->capacity());
?>

以上例程的输出类似于：

int(16)
