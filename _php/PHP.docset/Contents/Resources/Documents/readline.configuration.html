运行时配置

这些函数的行为受 php.ini 中的设置影响。

Readline 配置选项
名字 默认 可修改范围 更新日志
cli.pager "" PHP_INI_ALL 自 PHP 5.4.0 可用.
cli.prompt "\\b \\> " PHP_INI_ALL 自 PHP 5.4.0 可用.

这是配置指令的简短说明。

cli.pager string

External tool to display output from command line.

cli.prompt string

命令行 提示.
