ReflectionExtension::getName

(PHP 5, PHP 7, PHP 8)

ReflectionExtension::getName — 获取扩展名称

说明

public ReflectionExtension::getName(): string

获取扩展名称。

参数

此函数没有参数。

返回值

扩展名称。

范例

示例 #1 ReflectionExtension::getName() example

<?php
$ext = new ReflectionExtension('mysqli');
var_dump($ext->getName());
?>

以上例程的输出类似于：

string(6) "mysqli"
