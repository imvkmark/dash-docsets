fastcgi_finish_request

(PHP 5 >= 5.3.3, PHP 7, PHP 8)

fastcgi_finish_request — 冲刷(flush)所有响应的数据给客户端

说明

fastcgi_finish_request(): bool

此函数冲刷(flush)所有响应的数据给客户端并结束请求。 这使得客户端结束连接后，需要大量时间运行的任务能够继续运行。

返回值

成功时返回 true， 或者在失败时返回 false。
