ReflectionClass::getTraits

(PHP 5 >= 5.4.0, PHP 7, PHP 8)

ReflectionClass::getTraits — 返回这个类所使用的 traits 数组

说明

public ReflectionClass::getTraits(): array

警告

本函数还未编写文档，仅有参数列表。

参数

此函数没有参数。

返回值

返回了一个数组，键是 trait 的名称，值是 trait 实例的 ReflectionClass。 出现错误的情况下返回 null。
