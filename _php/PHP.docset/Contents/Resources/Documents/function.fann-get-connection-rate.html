fann_get_connection_rate

(PECL fann >= 1.0.0)

fann_get_connection_rate — 获取当网络创建时连接的使用率。

说明

fann_get_connection_rate(resource $ann): float

获取当网络创建时连接的使用率。

参数

ann

神经网络 资源。

返回值

成功，则返回当网络创建时连接的使用率, 错误则返回 false .
