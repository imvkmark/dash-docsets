pht\Vector::size

(PECL pht >= 0.0.1)

pht\Vector::size — Gets the size of the vector

说明

public pht\Vector::size(): int

Returns the current size of the vector. This operation requires a pht\Vector's mutex lock to be held if it is being used by multiple threads.

参数

此函数没有参数。

返回值

The size of the vector.

范例

示例 #1 Getting a vector's size

<?php

use pht\Vector;

$vector = new Vector();

$vector[] = 1;
$vector[] = 2;
$vector[] = 3;

var_dump($vector->size());

以上例程会输出：

int(3)
