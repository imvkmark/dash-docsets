apcu_enabled

(PECL apcu >= 4.0.3)

apcu_enabled — Whether APCu is usable in the current environment

说明

apcu_enabled(): bool

Returns whether APCu is usable in the current environment.

参数

此函数没有参数。

返回值

Returns true when APCu is usable in the current environment, false otherwise.
